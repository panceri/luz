# Luz

Apenas um desafio proposto pela empresa em questão.


## Installation

Clonar o repositório

```ruby
bundle
rake install
luz coupons.csv products.csv orders.csv order_items.csv outuput.csv
```

Para executar os testes:

    $ rspec
